```bash
git clone https://github.com/manavpatnaik/easyshop.git
```


<h3>Starting</h3>

To build and run the JAR

```bash
cd easyshop
mvn clean package -DskipTests
java -jar target/easyshop-0.0.1-SNAPSHOT.jar
```
